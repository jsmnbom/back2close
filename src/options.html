If not checked, Back to Close will only load on tabs with a parent (opened via "open in new tab").

When checked, Back to Close WE will briefly change the title to "Close tab" when injecting a history state. This shows when right clicking on the back button. This may lead to a tab getting stuck with this title, if this is unacceptable, uncheck this option.

Tampermonkey became with version 4.15.6154 incompatible with Back to Close. When checked, Back to Close will work with Tampermonkey 4.15.6154 and later, but it will not work without it.
